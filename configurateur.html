Configuration du sous-sol
Le type de sous-sol détermine la séquence de démarrage du GO
🏗️
Sous-sol général
Un seul sous-sol sous tous les bâtiments. Les bâtiments ne démarrent qu'une fois le sous-sol terminé.
🏢🏢
Sous-sols partiels
Certains bâtiments ont un sous-sol, d'autres non. Les bâtiments sans sous-sol démarrent indépendamment.
🏠
Pas de sous-sol
Tous les bâtiments sur terre-plein. Fondations directes puis élévations.
Sous-sol général — Séquence obligatoire avant les superstructures :
Terrassement → Voiles contre terre → Dalle SS-n → Élév. SS-n → ... → Dalle RDC
→ Puis enchaînement bâtiment par bâtiment en élévation RDC.

⚠ Le sous-sol est un verrou : aucun bâtiment ne peut démarrer en superstructure tant que la dalle RDC de sa zone n'est pas coulée.

Les bâtiments sans sous-sol peuvent démarrer leurs fondations et élévations indépendamment du sous-sol. Les bâtiments avec sous-sol suivent la séquence : terrassement → voiles CT → dalles → puis élévation.
Pas de sous-sol — Chaque bâtiment démarre directement par :
Terrassement → Fondations → Dallage → Élévation RDC → R+1...
Le phasage inter-bâtiments reste paramétrable (décalage entre bâtiments).
